Getting Started

This Flask boilerplate provides a solid foundation for building modern web applications.

Prerequisites

  • Python 3.9+
  • PostgreSQL 14+ (or SQLite for development)
  • Node.js 18+ (for Tailwind CSS)

Installation

# Create virtual environment
python -m venv venv
source venv/bin/activate  # On Windows: venv\Scripts\activate

# Install dependencies
pip install -r requirements.txt
npm install

# Set up environment
cp env.example .env
# Edit .env with your configuration

# Initialize database
flask db upgrade

# Build frontend assets
npm run build

# Start development server
python run.py

Configuration

The application uses environment variables for configuration. Copy env.example to .env and modify as needed.

Key Settings

  • FLASK_ENV - Environment (development, production)
  • DATABASE_URL - Database connection string
  • SECRET_KEY - Flask secret key
  • JWT_SECRET_KEY - JWT signing key

Development

The boilerplate includes comprehensive development tools and testing setup.

Available Commands

# Run tests
pytest

# Run tests with coverage
pytest --cov=app

# Format code
black .
isort .

# Database migrations
flask db migrate -m "Description"
flask db upgrade

# Frontend development
npm run dev  # Watch mode
npm run build  # Production build

Deployment

The application is production-ready with proper configuration for deployment.

Production Settings

  • Set FLASK_ENV=production
  • Use a production database (PostgreSQL recommended)
  • Set secure secret keys
  • Configure proper logging
  • Use a production WSGI server (Gunicorn)
