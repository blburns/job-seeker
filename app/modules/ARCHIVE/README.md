# Archived modules

Unregistered boilerplate modules kept for reference only. They are **not**
loaded by the application factory.

| Path | Origin |
|------|--------|
| `crm/` | Legacy CRM routes/API stubs from the CRM boilerplate |
| `account/` | Vuexy account-settings demo pages (not job-seeker settings) |
| `accounts/`, `network/`, `profile/`, `security/`, `store/` | Earlier boilerplate demos |

Do not import these packages from active code. Job seeker features live under
`resume/`, `jobs/`, `applications/`, `apply/`, and `analytics/`.
